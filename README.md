# 🏰 Wealth Tower

> Personal wealth management & investment intelligence platform
> A major module inside BCC (Brain Command Centre)

## 🎯 Mission

Build for normal people, not professional traders. Help users make better financial decisions and grow wealth over time through:

- 📊 Real-time portfolio tracking (via Tiger Brokers Open API)
- 🎯 Goal tracking with feasibility analysis
- ⚡ Smart Signals (Buy / Hold / Reduce — no jargon)
- 🔍 Opportunity Radar
- 🚀 IPO Command Centre
- 💰 Passive income tracking

## 🛡️ Safety First

- **Phase 1:** Read-only Tiger API integration. No trading.
- All API keys server-side only. Never exposed to frontend.
- Environment variables only. Never hardcoded secrets.

## 🏗️ Tech Stack

- **Backend:** Python (Tiger SDK) + Node.js
- **Database:** Supabase (Singapore region)
- **Frontend:** Next.js + React + Tailwind CSS
- **Hosting:** Vercel
- **Broker:** Tiger Brokers (Prime Account read-only)
